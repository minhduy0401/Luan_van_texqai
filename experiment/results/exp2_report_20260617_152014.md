- **ACC₂LLM**: Nếu ≥ 2 trong 3 LLM phân loại đúng mức Bloom → hệ thống đáng tin cậy.

- Model đánh giá dùng prompt zero-shot, temperature=0 để đảm bảo tính nhất quán.
