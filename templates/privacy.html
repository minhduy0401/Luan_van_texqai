1. {{ 'Introduction' if current_lang == 'en' else 'Giới thiệu' }}
{% if current_lang == 'en' %}

Welcome to TEXTQAI — an automatic question generation system based on Bloom's taxonomy. We are committed to protecting your privacy. This policy describes how we collect, use, and protect your personal information when you use our application.

{% else %}

Chào mừng bạn đến với TEXTQAI — hệ thống sinh câu hỏi tự động theo thang Bloom. Chúng tôi cam kết bảo vệ quyền riêng tư của bạn. Chính sách này mô tả cách chúng tôi thu thập, sử dụng và bảo vệ thông tin cá nhân khi bạn sử dụng ứng dụng.

{% endif %}
2. {{ 'Information We Collect' if current_lang == 'en' else 'Thông tin chúng tôi thu thập' }}
    {% if current_lang == 'en' %}
  • Account information: email, display name, encrypted password (for standard registration) or Google OAuth profile (for Google sign-in).
  • Uploaded documents: PDF files and the extracted text. This data is only used for question generation and is never shared.
  • Q&A results: question-answer pairs, Bloom level, and quality scores generated by the system.
  • Transaction history: order codes, purchased packages, payment status (no bank card data is stored).
  • Feedback: email and message content submitted via the contact form.
  • System logs: anonymous technical logs (errors, processing time) to improve the service.
    • {% else %}
  • Thông tin tài khoản: email, tên hiển thị, mật khẩu đã mã hoá (khi đăng ký thông thường) hoặc thông tin Google OAuth (khi đăng nhập bằng Google).
  • Tài liệu người dùng upload: file PDF và văn bản trích xuất. Dữ liệu này chỉ dùng để sinh câu hỏi và không được chia sẻ.
  • Kết quả sinh câu hỏi: các cặp Q&A, cấp độ Bloom, điểm chất lượng do hệ thống tính toán.
  • Lịch sử giao dịch: mã đơn hàng, gói đã mua, trạng thái thanh toán (không lưu thông tin thẻ ngân hàng).
  • Phản hồi (Feedback): email và nội dung tin nhắn bạn gửi qua form liên hệ.
  • Nhật ký hệ thống: log kỹ thuật ẩn danh (lỗi, thời gian xử lý) để cải thiện dịch vụ.
    • {% endif %}
3. {{ 'Purpose of Data Use' if current_lang == 'en' else 'Mục đích sử dụng dữ liệu' }}
    {% if current_lang == 'en' %}
  • To provide and operate the automatic question generation feature.
  • To manage accounts and authenticate users.
  • To process payments and track credit balances.
  • To improve service quality through technical log analysis.
  • To respond to user support requests.
    • {% else %}
  • Cung cấp và vận hành tính năng sinh câu hỏi tự động.
  • Quản lý tài khoản và xác thực danh tính người dùng.
  • Xử lý thanh toán và theo dõi số dư credits.
  • Cải thiện chất lượng dịch vụ thông qua phân tích log kỹ thuật.
  • Phản hồi yêu cầu hỗ trợ của người dùng.
    • {% endif %}
4. {{ 'Third-Party Data Sharing' if current_lang == 'en' else 'Chia sẻ dữ liệu với bên thứ ba' }}
{% if current_lang == 'en' %}

We do not sell your personal information. Data is only transferred through the following parties:

  • Google OAuth 2.0: when you choose to sign in with Google, Google authenticates your identity and returns basic profile information.
  • OpenRouter / LLM API: chapter/section content from your document is sent to the API for question generation. We use providers' data non-retention terms.
  • Sepay - VNPay: the payment gateway processes transactions. Bank/card data is managed by Sepay - VNPay and never passes through our servers.
{% else %}

Chúng tôi không bán thông tin cá nhân của bạn. Dữ liệu chỉ được chuyển qua các bên sau:

  • Google OAuth 2.0: khi bạn chọn đăng nhập bằng Google, Google xác thực danh tính và trả về thông tin hồ sơ cơ bản.
  • OpenRouter / LLM API: nội dung chương/phần từ tài liệu của bạn được gửi đến API để sinh câu hỏi. Chúng tôi sử dụng điều khoản không lưu trữ dữ liệu của các nhà cung cấp này.
  • Sepay - VNPay: cổng thanh toán xử lý giao dịch. Dữ liệu thẻ/tài khoản ngân hàng do Sepay - VNPay quản lý, không qua server của chúng tôi.
{% endif %}
5. {{ 'Data Security' if current_lang == 'en' else 'Bảo mật dữ liệu' }}
    {% if current_lang == 'en' %}
  • Passwords are encrypted using the bcrypt algorithm (Werkzeug) before being stored in the database.
  • Connections use HTTPS/TLS.
  • API keys and sensitive information are stored in environment variables, not in source code.
    • {% else %}
  • Mật khẩu được mã hoá bằng thuật toán bcrypt (Werkzeug) trước khi lưu vào cơ sở dữ liệu.
  • Kết nối sử dụng HTTPS/TLS.
  • Khoá API và thông tin nhạy cảm lưu trong biến môi trường, không lưu trong mã nguồn.
    • {% endif %}
