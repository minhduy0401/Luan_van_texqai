AI Usage Policy — TEXTQAI

{{ 'Last updated' if current_lang == 'en' else 'Cập nhật lần cuối' }}: {{ last_updated }}
1. {{ 'Purpose' if current_lang == 'en' else 'Mục đích' }}
{% if current_lang == 'en' %}

TEXTQAI uses large language models (LLMs) to analyze documents, generate exam questions and answers aligned with Bloom's taxonomy, validate output quality, and optionally translate Q&A to match your interface language.

{% else %}

TEXTQAI sử dụng mô hình ngôn ngữ lớn (LLM) để phân tích tài liệu, sinh câu hỏi và đáp án theo thang Bloom, kiểm tra chất lượng đầu ra, và (khi cần) dịch Q&A sang ngôn ngữ giao diện của bạn.

{% endif %}
2. {{ 'How AI Is Used' if current_lang == 'en' else 'AI được dùng như thế nào' }}
    {% if current_lang == 'en' %}
  • Document analysis: chapter/section structure is inferred from extracted PDF text.
  • Question generation (Agent 2): section content is sent to the configured LLM to produce questions and answers.
  • Quality validation (Agent 3): generated Q&A is checked against source content and Bloom-level rules.
  • Translation (optional): when document language differs from UI language, Q&A may be translated by the same LLM provider.
    • {% else %}
  • Phân tích tài liệu: cấu trúc chương/mục được suy luận từ văn bản trích xuất từ PDF.
  • Sinh câu hỏi (Agent 2): nội dung mục được gửi tới LLM đã cấu hình để tạo câu hỏi và đáp án.
  • Kiểm tra chất lượng (Agent 3): Q&A được đối chiếu với nguồn và quy tắc mức Bloom.
  • Dịch thuật (tùy chọn): khi ngôn ngữ tài liệu khác ngôn ngữ giao diện, Q&A có thể được LLM dịch sang ngôn ngữ tương ứng.
    • {% endif %}
3. {{ 'Third-Party Providers' if current_lang == 'en' else 'Nhà cung cấp bên thứ ba' }}
{% if current_lang == 'en' %}

Depending on admin configuration, requests may be routed through OpenRouter, OpenAI, or Google Gemini APIs. Only the minimum text needed for each step is transmitted. We select providers that offer data non-retention or limited-retention terms where available.

{% else %}

Tùy cấu hình admin, yêu cầu có thể đi qua API OpenRouter, OpenAI hoặc Google Gemini. Chỉ gửi phần văn bản tối thiểu cần thiết cho từng bước. Chúng tôi ưu tiên nhà cung cấp có điều khoản không lưu trữ hoặc lưu trữ hạn chế dữ liệu khi có thể.

{% endif %}
4. {{ 'Your Responsibilities' if current_lang == 'en' else 'Trách nhiệm của bạn' }}
    {% if current_lang == 'en' %}
  • Upload only documents you have the right to use for educational or research purposes.
  • Do not upload unlawful, confidential, or personal data of third parties without consent.
  • Review AI-generated output before use in exams or official assessments.
    • {% else %}
  • Chỉ upload tài liệu bạn có quyền sử dụng cho mục đích giáo dục hoặc nghiên cứu.
  • Không upload nội dung vi phạm pháp luật, bí mật hoặc dữ liệu cá nhân của bên thứ ba khi chưa có sự đồng ý.
  • Kiểm tra lại kết quả do AI sinh trước khi dùng cho thi cử hoặc đánh giá chính thức.
    • {% endif %}
5. {{ 'Limitations & Disclaimer' if current_lang == 'en' else 'Giới hạn & miễn trừ' }}
{% if current_lang == 'en' %}

AI output may contain inaccuracies, omissions, or misclassified Bloom levels. TEXTQAI provides results on an "as-is" basis for academic assistance — not as a substitute for professional or legal advice. Service availability depends on third-party API uptime and rate limits.

{% else %}

Kết quả AI có thể sai sót, thiếu ý hoặc phân loại Bloom chưa chính xác. TEXTQAI cung cấp kết quả theo dạng "nguyên trạng" để hỗ trợ học thuật — không thay thế tư vấn chuyên môn hay pháp lý. Dịch vụ phụ thuộc thời gian hoạt động và giới hạn tốc độ của API bên thứ ba.

{% endif %}
