{% if current_lang == 'en' %}

This policy describes how payments, credits, and subscriptions work on TEXTQAI. By purchasing credits or packages, you agree to the terms below.

2. {{ 'Payment Methods' if current_lang == 'en' else 'Phương thức thanh toán' }}
    {% if current_lang == 'en' %}
  • VNPay: online payment via the VNPay gateway (sandbox or production, depending on system configuration).
  • Bank transfer: manual transfer to the account displayed on the payment page; credits are added after admin or webhook confirmation.
  • SePay webhook: automatic confirmation when a transfer matches the order reference.
    • {% else %}
  • VNPay: thanh toán trực tuyến qua cổng VNPay (sandbox hoặc production tùy cấu hình hệ thống).
  • Chuyển khoản ngân hàng: chuyển khoản thủ công theo thông tin trên trang thanh toán; credits được cộng sau khi admin hoặc webhook xác nhận.
  • SePay webhook: xác nhận tự động khi giao dịch khớp mã đơn hàng.
    • {% endif %}
3. {{ 'Credits & Packages' if current_lang == 'en' else 'Credits & Gói dịch vụ' }}
{% if current_lang == 'en' %}

Credits are consumed when you generate question sets. Package prices and credit amounts are shown on the Pricing page before checkout. Successfully added credits are tied to your account and cannot be transferred to another user.

{% else %}

Credits được trừ khi bạn sinh bộ câu hỏi. Giá gói và số credits hiển thị tại trang Bảng giá trước khi thanh toán. Credits đã nạp thành công gắn với tài khoản của bạn và không chuyển nhượng cho người khác.

{% endif %}
4. {{ 'Refunds' if current_lang == 'en' else 'Hoàn tiền' }}
{% if current_lang == 'en' %}

Successfully purchased credits are non-refundable, except when a confirmed system error caused incorrect charging or failed credit delivery. We do not refund credits already used for generation.

{% else %}

Credits đã mua thành công không hoàn tiền, trừ khi có lỗi hệ thống được xác nhận khiến bạn bị trừ tiền sai hoặc không nhận đủ credits. Credits đã sử dụng để sinh câu hỏi không được hoàn lại.

{% endif %}
5. {{ 'Transaction Records' if current_lang == 'en' else 'Lưu trữ giao dịch' }}
{% if current_lang == 'en' %}

We store order codes, package names, amounts, payment status, and timestamps. We do not store full bank card numbers on our servers — card/banking data is handled by VNPay or your bank.

{% else %}

Chúng tôi lưu mã đơn, tên gói, số tiền, trạng thái thanh toán và thời gian giao dịch. Chúng tôi không lưu số thẻ ngân hàng đầy đủ trên server — dữ liệu thẻ/tài khoản do VNPay hoặc ngân hàng xử lý.

{% endif %}
